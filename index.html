IIIF Demo — AI + History Collaboratory

Static IIIF image tiles served from GitHub Pages. Prepared for the Collaboratory session on 24 February 2026: "Exploring MCP as an Interface with Databases."

View in Mirador

Copy the manifest URL below and paste it into the Mirador demo viewer:

https://ai-and-history-collaboratory.github.io/iiif-demo/manifests/demo-manifest.json

How to open:

1. Go to projectmirador.org/demo
2. Click the + (add resource) button in the top left
3. Paste the manifest URL above into the input field and click Add
4. Select the image to open — you can zoom in and out to explore the manuscript

Images in this Demo

Takvim-i Vekayi, No. 1 (1 November 1831) — First page of the first issue of the Ottoman Government Gazette, published in Istanbul. Source: Wikimedia Commons. Public Domain.

How This Was Made

The source image (13,075 × 9,129 pixels) was downloaded from Wikimedia Commons, downscaled to 5,000 × 3,491 pixels, and converted into a pyramid of 256×256 pixel IIIF tiles at five zoom levels using Claude Cowork and the Python iiif library. The tiles, info.json, and IIIF Presentation API v3 manifest were generated programmatically, then pushed to this GitHub Pages site. No image server is required — the entire setup is static files.
